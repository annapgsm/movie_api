Add Favorite Movie Adds a movie title string to a user's favoriteMovies array POST /users/:id/:movieTitle :id (number, required), :movieTitle (string, required) None

Remove Favorite Movie Removes a movie title from a user's favoriteMovies array DELETE /users/:id/:movieTitle :id (number, required), :movieTitle (string, required) None 
    "Psycho has been removed from user 2's array"
Delete User Removes a user by ID from the system DELETE /users/:id :id (number, required) None
