Add Favorite Movie Adds a movie reference to a user's favoriteMovies array POST /users/:id/favorites/:movieId :id (string, required), :movieId (string, required) None 
"Movie has been added to user's favorites"
Remove Favorite Movie Removes a movie reference from a user's favoriteMovies array DELETE /users/:id/favorites/:movieId :id (string, required), :movieId (string, required) None 
"Movie has been removed from user's favorites"
Delete User Removes a user by ID DELETE /users/:id :id (string, required) None 
"User has been deleted"
